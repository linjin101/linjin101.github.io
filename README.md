# linjin101.github.io  

程序猿的空间代码分支hexo存放博客代码,master存放博客文件  
需要修改内容或者写文章,代码克隆岛hexo分支下面,发布到master主分支  

*本地获取代码->选择"hexo"分支,以下命令会把生成的html自动提交到master分支

#####一次性执行:清空、刷新、部署三个命令  

``hexo clean && hexo g && hexo d``

####本地测试运行  
``` hexo s  ```
